# cpp-shell

A utility of `shell`.

## Requirements

cpp-shell depends the following classes.

- [pqrs::string](https://github.com/pqrs-org/cpp-string)

## Install

Copy `include/pqrs` and `vendor/vendor/include` directories into your include directory.
